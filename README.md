# World Models, Reasoning, and Interpretable Decision Systems

*How do modern learning systems build representations of the world, reason over them and act robustly under uncertainty?*

A world model is an internal learned representation of reality that acts as the "imagination" of a decision making agent. Engineering a world model can be formidable: they are typically composed of multiple function approximators working in tandem. 
My goal with this repository is to slowly and steadily break apart the idea of a world model into the following steps:
1. Representation 
2. Prediction
3. Control
4. Reasoning 
5. Interpretation 
6. Alignment
